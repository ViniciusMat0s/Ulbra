Semipresencial Design Patterns

Vinícius Matos

O Pattern utilizado foi o Factory Method, implementado na linguagem JavaScript.

Utilizei o Factory para simular o trabalho do RH de uma empresa, mais especificamente, o Hotel em que trabalho.

Explicação do código

1. Uma função para cada cargo na empresa, informando nome e o próprio cargo;

2. A função "RH" define a construção de cada um dos cargos, mostrando as informações passadas acima utilizando o "switch case";

3. A função "apresentação" cria uma mensagem de apresentação utilizando os parâmetros do RH;

4. São criadas as constantes "rh" para inserir o nome e o cargo, utilizando os números do switch case e "funcionarios" para criar a lista de funcionários, abaixo, suas respectivas inserções;

5. Por fim, o "forEach" tem o trabalho de apresentar todos os membros da equipe.
